# Fauez Passos - Personal Links 🚀

Welcome to my personal links page! This is a simple, modern, and fully responsive "linktree" style landing page built with pure HTML, CSS, and vanilla JavaScript.

## Features ✨
- **Glassmorphism Design:** Beautiful frosted glass effects using modern CSS properties.
- **Dynamic Animated Background:** Organic, moving background shapes that bring the page to life.
- **Interactive 3D Hover:** The central card reacts to your mouse movements (on desktop).
- **Responsive:** Looks great on mobile devices and large monitors.
- **Lightweight:** No heavy frameworks, no external image icons. Built with inline SVGs for maximum performance.

## Tech Stack 🛠️
- HTML5
- CSS3 (Custom properties, animations, flexbox)
- JavaScript (Vanilla DOM manipulation and 3D math)

## Setup 💻
To view the project, simply clone the repository and open `index.html` in any modern web browser. No complicated build steps required!

```bash
git clone https://github.com/faopassos/faopassos.github.io.git
cd faopassos.github.io
# Open index.html in your browser
```
